# Project Setup Instructions

## Running the Website Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Rohan89283/Projecttest3948.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Projecttest3948
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to view the website.

## Deploying to Railway
1. Sign in to your Railway account at [Railway](https://railway.app).
2. Click on "New Project".
3. Choose "Deploy from GitHub".
4. Select the `Projecttest3948` repository from your GitHub account.
5. Configure build settings if necessary, and click on "Deploy".
6. Once deployed, Railway will provide you with a link to access your live website!
